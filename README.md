# Smart Expense Tracker

A Python-based expense tracking application that helps users record, categorize, and analyze their daily expenses intelligently.

## Features (Planned)
- Add, edit, and delete expenses
- Auto-categorization of transactions
- Visual analytics (charts and summaries)
- Data export (CSV/Excel)
- Optional AI assistant for spending insights

## Setup
1. Clone the repository
2. Run `pip install -r requirements.txt`
3. Run `python main.py`
